# Thunder Client

Thunder Client is a VS Code extension for API testing. This guide covers Thunder Client features, usage, and best practices.

## Thunder Client Overview

Thunder Client provides:
- **VS Code Integration**: Integrated in VS Code
- **API Testing**: REST API testing
- **Collections**: Request collections
- **Environments**: Environment support
- **Lightweight**: Lightweight extension

## Getting Started

### Installation

1. **Open VS Code**: Open VS Code
