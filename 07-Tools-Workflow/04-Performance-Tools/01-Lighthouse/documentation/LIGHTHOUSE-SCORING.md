# Lighthouse Scoring

Lighthouse scoring provides performance ratings. This guide covers score calculation, interpretation, and improvement strategies.

## Score Overview

Lighthouse provides scores for:
- **Performance**: 0-100 performance score
- **Accessibility**: 0-100 accessibility score
- **Best Practices**: 0-100 best practices score
- **SEO**: 0-100 SEO score

## Performance Score

### Score Calculation

Performance score is calculated from:
- **First Contentful Paint**: FCP metric
- **Largest Contentful Paint**: LCP metric
- **Total Blocking Time**: TBT metric
- **Speed Index**: Speed Index
- **Cumulative Layout Shift**: CLS metric

### Score Ranges

- **90-100**: Excellent (Green)
- **50-89**: Needs Improvement (Orange)
- **0-49**: Poor (Red)

### Improving Score

1. **Optimize Metrics**: Improve Core Web Vitals
2. **Reduce JavaScript**: Minimize JavaScript
3. **Optimize Images**: Optimize images
4. **Caching**: Implement caching
5. **Code Splitting**: Split code

## Accessibility Score

### Score Calculation

Accessibility score measures:
- **ARIA Usage**: Proper ARIA usage
- **Color Contrast**: Color contrast
- **Keyboard Navigation**: Keyboard accessibility
- **Screen Reader**: Screen reader compatibility

### Improving Score

1. **ARIA Labels**: Add ARIA labels
2. **Color Contrast**: Improve contrast
3. **Keyboard Navigation**: Ensure keyboard access
4. **Semantic HTML**: Use semantic HTML
5. **Alt Text**: Add alt text

## Best Practices Score

### Score Calculation

Best Practices score checks:
- **HTTPS**: HTTPS usage
- **Console Errors**: JavaScript errors
- **Image Aspect Ratio**: Image dimensions
- **Deprecated APIs**: Deprecated API usage

### Improving Score

1. **HTTPS**: Use HTTPS
2. **Fix Errors**: Fix JavaScript errors
3. **Set Dimensions**: Set image dimensions
4. **Update APIs**: Update deprecated APIs
5. **Security**: Follow security best practices
