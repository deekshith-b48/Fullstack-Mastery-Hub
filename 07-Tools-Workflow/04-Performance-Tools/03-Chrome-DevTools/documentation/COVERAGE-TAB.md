# Coverage Tab

The Coverage tab shows unused code. This guide covers code coverage analysis, optimization strategies, and best practices.

## Coverage Overview

Coverage tab provides:
- **Unused Code**: Identify unused code
- **CSS Coverage**: Unused CSS detection
- **JavaScript Coverage**: Unused JavaScript detection
- **Optimization**: Code optimization insights
- **Bundle Size**: Reduce bundle size

## Using Coverage

### Starting Coverage

1. **Open Drawer**: Open DevTools drawer
2. **Coverage Tab**: Click Coverage tab
3. **Start**: Click record button
4. **Interact**: Interact with page
5. **Stop**: Stop recording
6. **Analyze**: Analyze results

### Coverage Results

- **URL**: Resource URL
- **Type**: Resource type (CSS/JS)
- **Total Bytes**: Total size
- **Unused Bytes**: Unused size
- **Usage**: Usage percentage

## Analyzing Coverage

### JavaScript Coverage

- **Unused Functions**: Unused functions
- **Unused Code**: Unused code blocks
- **Dead Code**: Dead code elimination
- **Tree Shaking**: Tree shaking opportunities
- **Code Splitting**: Code splitting opportunities

### CSS Coverage

- **Unused Styles**: Unused CSS rules
- **Unused Selectors**: Unused selectors
- **Unused Classes**: Unused CSS classes
- **Media Queries**: Unused media queries
- **Removed Styles**: Styles to remove
