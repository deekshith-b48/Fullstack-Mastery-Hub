# Layout Panel

The Layout panel inspects layout and CSS. This guide covers layout inspection, CSS grid, flexbox, and layout debugging.

## Layout Panel Overview

Layout panel provides:
- **Layout Inspection**: Inspect layout properties
- **Grid Overlay**: CSS Grid overlay
- **Flexbox Overlay**: Flexbox overlay
- **Box Model**: Box model visualization
- **Computed Styles**: Computed style values

## Grid Overlay

### CSS Grid Inspection

1. **Select Element**: Select grid container
2. **Enable Overlay**: Enable grid overlay
3. **View Grid**: Visualize grid lines
4. **Track Sizes**: View track sizes
5. **Gap**: View grid gaps

### Grid Features

- **Grid Lines**: Visualize grid lines
- **Track Sizes**: Track size information
- **Gap Visualization**: Gap visualization
- **Area Names**: Named grid areas
- **Grid Items**: Grid item positioning

## Flexbox Overlay

### Flexbox Inspection

1. **Select Element**: Select flex container
2. **Enable Overlay**: Enable flexbox overlay
3. **View Flex**: Visualize flex properties
4. **Axis**: View main/cross axis
5. **Items**: View flex items

### Flexbox Features

- **Direction**: Flex direction
- **Justify Content**: Justify content visualization
- **Align Items**: Align items visualization
- **Flex Properties**: Flex item properties
- **Gap**: Flex gap visualization

## Box Model

### Box Model Visualization

- **Content**: Content box
- **Padding**: Padding area
- **Border**: Border area
- **Margin**: Margin area
- **Dimensions**: Element dimensions

### Computed Styles

- **All Properties**: All computed properties
- **Filter**: Filter properties
- **Values**: Computed values
- **Inherited**: Inherited properties
- **Overridden**: Overridden properties
