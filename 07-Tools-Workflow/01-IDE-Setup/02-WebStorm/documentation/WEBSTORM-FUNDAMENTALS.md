# WebStorm Fundamentals

WebStorm is a powerful IDE for JavaScript, TypeScript, and web development. Developed by JetBrains, it provides comprehensive tools for modern web development.

## What is WebStorm?

WebStorm is a commercial IDE specifically designed for JavaScript and web development. It offers:
- Intelligent code completion
- Built-in debugging
- Version control integration
- Refactoring tools
- Testing frameworks support
- Database tools

## Key Features

### Code Intelligence

- **Smart Code Completion**: Context-aware suggestions
- **On-the-fly Error Detection**: Real-time error checking
- **Code Navigation**: Jump to definitions, usages
- **Refactoring**: Safe code transformations
- **Code Inspection**: Code quality analysis

### Built-in Tools

- **Debugger**: JavaScript and Node.js debugging
- **Terminal**: Integrated terminal
- **Version Control**: Git, SVN, Mercurial support
- **Database Tools**: SQL databases support
- **HTTP Client**: Built-in REST client
- **Test Runner**: Jest, Mocha, Karma support

### Framework Support

- **React**: Full React support
- **Vue.js**: Vue 2 and Vue 3 support
- **Angular**: Complete Angular support
- **Node.js**: Node.js development tools
- **TypeScript**: Full TypeScript support

## Installation

### System Requirements

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.14 or later
- **Linux**: Any modern Linux distribution
- **RAM**: 4GB minimum, 8GB recommended
- **Disk Space**: 3GB for installation

### Download and Install

1. Visit jetbrains.com/webstorm
2. Download installer
3. Run installer
4. Follow installation wizard
5. Launch WebStorm

### First Launch

On first launch:
1. Choose UI theme (Light/Dark)
2. Configure keymap (macOS/Windows/Linux)
3. Install plugins
4. Configure settings

## Project Setup

### Creating a Project

1. File > New > Project
2. Choose project type
3. Configure settings
4. Create project

### Opening Existing Project

1. File > Open
2. Select project folder
3. WebStorm opens project

## Essential Workflows

### Code Editing

- **IntelliSense**: Smart code completion
- **Live Templates**: Code snippets
- **Multiple Cursors**: Edit multiple locations
- **Code Formatting**: Auto-format code

### Navigation

- **Go to Definition**: `Ctrl+B` / `Cmd+B`
- **Go to Declaration**: `Ctrl+Alt+B` / `Cmd+Alt+B`
- **Find Usages**: `Alt+F7` / `Option+F7`
- **Recent Files**: `Ctrl+E` / `Cmd+E`

### Search

- **Find in Files**: `Ctrl+Shift+F` / `Cmd+Shift+F`
- **Replace in Files**: `Ctrl+Shift+R` / `Cmd+Shift+R`
- **Search Everywhere**: `Double Shift`
- **Find Usages**: `Alt+F7` / `Option+F7`
