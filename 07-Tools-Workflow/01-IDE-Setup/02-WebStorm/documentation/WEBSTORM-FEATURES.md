# WebStorm Features

WebStorm includes a comprehensive set of features designed to enhance productivity and code quality. This guide covers the key features that make WebStorm a powerful IDE.

## Code Intelligence

### Smart Code Completion

- **Context-Aware Suggestions**: Intelligent completions based on context
- **Type-Aware Completion**: Suggestions based on variable types
- **Import Assistance**: Auto-import suggestions
- **Chain Completion**: Complete method chains
- **Symbol Completion**: Complete class, method, and variable names

### Code Analysis

- **On-the-Fly Error Detection**: Real-time error highlighting
- **Code Inspections**: Static code analysis
- **Unused Code Detection**: Find unused variables, imports, functions
- **Code Duplication Detection**: Find duplicate code blocks
- **Performance Warnings**: Identify performance issues

### Navigation

- **Go to Definition**: Jump to symbol definitions
- **Go to Declaration**: Navigate to declarations
- **Find Usages**: Find all usages of a symbol
- **Type Hierarchy**: View class hierarchies
- **Call Hierarchy**: See function call chains

## Refactoring Tools

### Safe Refactoring

- **Rename**: Rename symbols safely across project
- **Extract**: Extract methods, variables, constants
- **Inline**: Inline variables, methods, constants
- **Move**: Move classes, methods to different locations
- **Change Signature**: Modify function signatures

### Code Transformations

- **Convert to Arrow Function**: Transform functions
- **Convert to Template String**: Convert string concatenation
- **Extract Component**: Extract React components
- **Introduce Variable**: Extract expressions to variables

## Framework Support

### React

- **JSX Support**: Full JSX syntax support
- **Component Navigation**: Navigate React components
- **Props Validation**: Validate component props
- **Hooks Support**: React Hooks support
- **Styled Components**: CSS-in-JS support

### Vue.js

- **Vue 2 and Vue 3**: Support for both versions
- **Single File Components**: .vue file support
- **Template Syntax**: Vue template highlighting
- **Vue Router**: Router support
- **Vuex/Pinia**: State management support

### Angular

- **TypeScript Integration**: Full TypeScript support
- **Angular CLI**: CLI integration
- **Component Templates**: Template support
- **Dependency Injection**: DI support
- **RxJS**: Reactive programming support
