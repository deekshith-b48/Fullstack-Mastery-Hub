# WebStorm Settings

WebStorm provides extensive customization options through settings. This guide covers configuration, preferences, and optimization settings.

## Accessing Settings

### Opening Settings

- **Windows/Linux**: `File > Settings` or `Ctrl+Alt+S`
- **macOS**: `WebStorm > Preferences` or `Cmd+,`
- **Quick Access**: Search for specific settings

### Settings Categories

- **Appearance & Behavior**: UI customization
- **Editor**: Code editor settings
- **Plugins**: Extension management
- **Build, Execution, Deployment**: Build and run configurations
- **Languages & Frameworks**: Language-specific settings
- **Tools**: External tools integration
- **Version Control**: VCS settings
