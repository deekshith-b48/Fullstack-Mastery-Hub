# Sublime Text Package Control

Package Control is the package manager for Sublime Text. It provides easy installation, updating, and management of packages (plugins) that extend Sublime Text's functionality.

## What is Package Control?

Package Control is:
- **Package Manager**: Install and manage packages
- **Repository**: Centralized package repository
- **Auto-Updates**: Automatic package updates
- **Dependency Management**: Handle package dependencies
- **Essential Tool**: Required for Sublime Text extensibility

## Installation

### Installing Package Control

1. Open Sublime Text
2. Open Command Palette: `Ctrl+Shift+P` / `Cmd+Shift+P`
3. Type "Install Package Control"
4. Press Enter
5. Wait for installation
6. Restart Sublime Text

### Manual Installation

1. Open Command Palette: `Ctrl+Shift+P` / `Cmd+Shift+P`
2. Type "Package Control: Install Package Control"
3. Or visit: packagecontrol.io/installation
4. Copy installation code
5. Paste into console: `Ctrl+` / `View > Show Console`

## Using Package Control

### Installing Packages

1. `Ctrl+Shift+P` / `Cmd+Shift+P`
2. Type "Package Control: Install Package"
3. Search for package name
4. Select package
5. Wait for installation

### Removing Packages

1. `Ctrl+Shift+P` / `Cmd+Shift+P`
2. Type "Package Control: Remove Package"
3. Select package to remove
4. Confirm removal

### Updating Packages

1. `Ctrl+Shift+P` / `Cmd+Shift+P`
2. Type "Package Control: Upgrade Package"
3. Select package to update
4. Or "Upgrade All Packages" to update all

### Listing Packages

1. `Ctrl+Shift+P` / `Cmd+Shift+P`
2. Type "Package Control: List Packages"
3. View installed packages

## Package Management

### Package Settings

- **Preferences** → **Package Settings** → **Package Name**
- Configure package settings
- Edit default or user settings

### Package Files

Packages are stored in:
- **Windows**: `%APPDATA%\Sublime Text 3\Packages\`
- **macOS**: `~/Library/Application Support/Sublime Text 3/Packages/`
- **Linux**: `~/.config/sublime-text-3/Packages/`

## Command Palette Commands

### Package Control Commands

- **Install Package**: Install new package
- **Remove Package**: Uninstall package

- **List Packages**: Show installed packages

## Best Practices

### Package Management

1. **Regular Updates**: Keep packages updated
2. **Minimal Packages**: Install only needed packages
3. **Review Packages**: Review installed packages regularly
4. **Backup Settings**: Backup package settings
5. **Document Packages**: Note why packages are installed

### Performance

1. **Limit Packages**: Too many packages slow startup
2. **Disable Unused**: Disable unused packages
3. **Monitor Performance**: Check package impact
4. **Remove Unused**: Remove packages you don't use

## Troubleshooting

### Package Issues

### Common Solutions

1. **Restart Sublime**: Restart after package changes
2. **Clear Cache**: Clear package cache
3. **Reinstall**: Reinstall problematic package
4. **Check Logs**: Review console for errors
