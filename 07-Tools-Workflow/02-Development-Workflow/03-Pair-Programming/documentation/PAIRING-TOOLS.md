# Pairing Tools

Pairing tools facilitate effective pair programming. This guide covers tools for remote pairing, code sharing, and collaboration.

## Tool Categories

### Video Communication

- **Zoom**: Video conferencing with screen share
- **Microsoft Teams**: Team collaboration platform
- **Google Meet**: Video meetings
- **Discord**: Voice and video chat

### Code Sharing

- **VS Code Live Share**: Real-time code sharing
- **CodeTogether**: Collaborative coding platform
- **Floobits**: Collaborative editing
- **Teletype**: Atom collaboration

### Pair Programming Platforms

- **Tuple**: Remote pair programming tool
- **CodeTogether**: Full-featured platform
- **VS Code Live Share**: VS Code integration
- **Floobits**: Multi-editor support

## VS Code Live Share

### Features

- **Real-time Editing**: Real-time code editing
- **Shared Terminal**: Shared terminal access
- **Shared Debugging**: Debug together
- **Audio/Video**: Built-in audio/video

### Setup

1. Install Live Share extension
2. Sign in with Microsoft/GitHub
3. Start collaboration session
4. Share session link

## CodeTogether

### Features

- **Multi-Editor**: Works with multiple editors
