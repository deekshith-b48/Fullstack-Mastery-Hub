# Jira Agile Boards

Jira Agile boards visualize work and manage sprints. This guide covers Scrum and Kanban boards, configuration, and usage for agile project management.

## Board Overview

Boards provide:
- **Visualization**: Visual work representation
- **Sprint Management**: Sprint planning and tracking
- **Backlog**: Prioritized backlog
- **Workflow**: Workflow visualization

## Board Types

### Scrum Board

- **Sprint Planning**: Sprint planning
- **Sprint Backlog**: Sprint backlog
- **Sprint Tracking**: Track sprint progress
- **Burndown**: Sprint burndown chart

### Kanban Board

- **Continuous Flow**: Continuous workflow
- **WIP Limits**: Work in progress limits
- **No Sprints**: No sprint structure
- **Flow Metrics**: Flow metrics
