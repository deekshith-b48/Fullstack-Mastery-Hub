# Trello Automation

Trello automation automates repetitive tasks. This guide covers Butler automation, rules, and best practices for efficient workflow automation.

## Automation Overview

Automation provides:
- **Task Automation**: Automate repetitive tasks
- **Workflow Automation**: Automate workflows
- **Rule-Based**: Rule-based automation
- **Time-Saving**: Save time

## Butler Automation

### Butler Features

- **Rules**: Create automation rules
- **Buttons**: Custom buttons

- **Scheduled**: Scheduled automation

### Rule Types

- **When-Then Rules**: When X happens, do Y
- **Card Actions**: Card-based automation
- **List Actions**: List-based automation
- **Board Actions**: Board-wide automation

## Automation Setup

### Creating Rules

1. **Access Butler**: Open Butler
2. **Create Rule**: Create new rule
3. **Define Trigger**: Define trigger
4. **Define Action**: Define action
5. **Save**: Save rule

### Rule Examples

**Auto-Move Cards:**
```
When a card is moved to "Done"
Then archive the card
```

**Auto-Assign:**
```
When a card is added to "In Progress"
Then assign to @username
```
