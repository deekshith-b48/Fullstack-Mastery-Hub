# Jira Projects

Jira projects organize work and issues. This guide covers project creation, configuration, and management for effective project organization.

## Project Overview

Projects provide:
- **Work Organization**: Organize work
- **Issue Tracking**: Track issues
- **Team Collaboration**: Team collaboration
- **Reporting**: Project reports

### Kanban Project

- **Kanban Board**: Kanban board
- **Continuous Flow**: Continuous flow
- **WIP Limits**: Work in progress limits
- **No Sprints**: No sprint structure

### Classic Project

- **Classic Workflow**: Traditional workflow
- **Custom Workflows**: Custom workflows
- **Flexibility**: Flexible structure
