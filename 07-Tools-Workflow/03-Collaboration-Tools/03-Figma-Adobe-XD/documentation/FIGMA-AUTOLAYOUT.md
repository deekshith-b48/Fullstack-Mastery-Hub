# Figma Autolayout

Figma Autolayout creates responsive, flexible layouts. This guide covers Autolayout features, usage, and best practices for modern design systems.

## Autolayout Overview

Autolayout provides:
- **Flexible Layouts**: Flexible, responsive layouts
- **Auto-Resizing**: Automatic resizing
- **Spacing**: Automatic spacing
- **Alignment**: Automatic alignment
- **CSS-Like**: Similar to CSS Flexbox

## Autolayout Features

### Layout Direction

- **Horizontal**: Left to right
- **Vertical**: Top to bottom
- **Flexible**: Flexible direction
- **Wrap**: Wrapping layout

### Spacing

- **Gap**: Space between items
- **Padding**: Internal padding
- **Auto-Spacing**: Automatic spacing
- **Consistent**: Consistent spacing

### Alignment

- **Alignment**: Align items
- **Distribution**: Distribute items
- **Justification**: Justify content
- **Auto-Alignment**: Automatic alignment
