# Figma Dev Mode

Figma Dev Mode provides developer-friendly features for design handoff. This guide covers Dev Mode features, usage, and best practices for seamless design-to-code workflows.

## Dev Mode Overview

Dev Mode provides:
- **Code Inspection**: Inspect design code
- **Specifications**: Design specifications
- **Export**: Asset export
- **Measurements**: Precise measurements
- **Code Generation**: CSS code generation

## Dev Mode Features

### Code Inspection

- **CSS Properties**: View CSS properties
- **Code Snippets**: Code snippets
- **Measurements**: Precise measurements
- **Colors**: Color values
- **Typography**: Typography specs

### Export

- **Asset Export**: Export assets
- **Multiple Formats**: Various formats
- **Optimization**: Optimized exports
- **Batch Export**: Batch export

### Measurements

- **Spacing**: Spacing measurements
- **Sizing**: Size measurements
- **Alignment**: Alignment guides
- **Grid**: Grid system

## Using Dev Mode

### Accessing Dev Mode

1. **Switch Mode**: Switch to Dev Mode
2. **Select Element**: Select design element
3. **View Specs**: View specifications
4. **Export**: Export assets
5. **Copy Code**: Copy code snippets

### Best Practices

1. **Design Review**: Review designs carefully
2. **Specs**: Use precise specifications
3. **Export**: Export optimized assets
4. **Communication**: Communicate with designers
5. **Documentation**: Document decisions
