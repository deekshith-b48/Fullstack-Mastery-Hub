# Notion Databases

Notion databases provide powerful data management. This guide covers database creation, views, and best practices for effective data organization.

## Database Overview

Databases provide:
- **Data Management**: Manage structured data
- **Multiple Views**: Different views
- **Relations**: Link databases
- **Filtering**: Filter and sort
- **Formulas**: Formula calculations

## Database Views

### Table View

- **Spreadsheet**: Spreadsheet-like view
- **Rows and Columns**: Rows and columns
- **Sorting**: Sort by columns
- **Filtering**: Filter rows

### Board View

- **Kanban**: Kanban board
- **Columns**: Status columns
- **Cards**: Database items as cards
- **Drag and Drop**: Move items

### Calendar View

- **Calendar**: Calendar layout
- **Date-based**: Date-based organization
- **Events**: Show items as events
- **Timeline**: Timeline view

### Gallery View

- **Visual**: Visual card layout
- **Images**: Show images
- **Card Preview**: Card previews
- **Grid Layout**: Grid arrangement
