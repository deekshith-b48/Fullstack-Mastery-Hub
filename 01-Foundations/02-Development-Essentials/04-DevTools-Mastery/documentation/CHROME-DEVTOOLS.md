# Chrome DevTools Guide

## Opening DevTools

- `F12` or `Ctrl+Shift+I` (Windows/Linux)
- `Cmd+Option+I` (Mac)
- Right-click â†’ Inspect

## Panels

### Elements Panel
- Inspect HTML/CSS
- Edit styles live
- Box model visualization

### Console Panel
- Run JavaScript
- View logs
- Debug errors

### Sources Panel
- Debug JavaScript
- Set breakpoints
- Step through code

### Network Panel
- Monitor requests
- View response headers
- Check loading times

### Application Panel
- Local Storage
- Session Storage
- Cookies
- IndexedDB

### Performance Panel
- Record performance
- Analyze bottlenecks
- CPU profiling

## Console Commands

```javascript
console.log('Message');
console.error('Error');
console.warn('Warning');
console.table(data);
console.time('label');
console.timeEnd('label');
```
