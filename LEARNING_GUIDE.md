- [ ] SQL basics
- [ ] Database design
- [ ] ORM/ODM (Sequelize/Prisma/Mongoose)

- [ ] Build: Full CRUD API with database

**Week 1-2: Connecting Frontend & Backend**
- [ ] API integration in frontend
- [ ] Authentication and authorization
- [ ] JWT tokens
- [ ] State management
- [ ] Build: Fullstack application with auth

**Week 3-4: Advanced Backend Topics**
- [ ] GraphQL basics
- [ ] File uploads

- [ ] Automated testing
- [ ] Deploy: Set up CI/CD

**Week 3-4: Deployment & DevOps**
- [ ] Docker basics
- [ ] Cloud platforms (AWS/Azure/GCP basics)
- [ ] Environment configuration
- [ ] Monitoring basics
- [ ] Deploy: Deploy your application

### Month 6: Advanced Topics & Portfolio

**Week 1-2: Testing & Quality**
- [ ] Unit testing
- [ ] Integration testing
- [ ] E2E testing
- [ ] Code quality tools
- [ ] Build: Add tests to your projects

**Week 3-4: Advanced Concepts**
- [ ] System design basics
- [ ] Performance optimization
- [ ] Security best practices
- [ ] Architecture patterns
- [ ] Build: Final portfolio project
